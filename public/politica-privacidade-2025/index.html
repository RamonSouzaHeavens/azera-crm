Política de Privacidade - Azera CRM

Última atualização: 20 de novembro de 2025

Quem somos

O Azera CRM é um sistema de gestão de relacionamento com clientes desenvolvido por Ramon Souza, sediado em Belo Horizonte/MG, Brasil. Nosso compromisso é fornecer uma plataforma eficiente e segura para gestão de leads, equipes e vendas, sempre respeitando a privacidade e os direitos dos usuários conforme a Lei Geral de Proteção de Dados (LGPD).

Quais dados coletamos

Coletamos os seguintes tipos de dados pessoais:

  • Informações de cadastro: nome, email, telefone, cargo e empresa
  • Dados de uso: informações sobre como você utiliza nossa plataforma
  • Dados técnicos: endereço IP, tipo de navegador, sistema operacional
  • Dados de leads: informações sobre prospects e clientes gerenciados no sistema

Como usamos seus dados

Utilizamos seus dados pessoais para:

  • Fornecer e melhorar nossos serviços
  • Personalizar sua experiência na plataforma
  • Enviar comunicações importantes sobre o serviço
  • Garantir a segurança e prevenir fraudes
  • Cumprir obrigações legais

Compartilhamento de dados

Não vendemos ou alugamos seus dados pessoais a terceiros. Podemos compartilhar informações apenas nas seguintes situações:

  • Com seu consentimento explícito
  • Para cumprir obrigações legais
  • Com provedores de serviços que nos ajudam a operar (sob contratos de confidencialidade)
  • Em caso de fusão, aquisição ou venda de ativos

Seus direitos sob a LGPD

De acordo com a Lei Geral de Proteção de Dados (LGPD), você tem os seguintes direitos:

  • Confirmação da existência de tratamento de dados
  • Acesso aos seus dados
  • Correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação de dados desnecessários
  • Portabilidade dos dados
  • Eliminação dos dados tratados com consentimento
  • Informação sobre compartilhamento de dados
  • Revogação do consentimento
